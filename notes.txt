
`nodemon` = **Node Monitor**
Ye ek **tool** hai jo tumhara backend (Node.js server) **automatically restart** karta hai **jab bhi tum file mein kuch change karo**.

---

### üß† Without `nodemon`:
Agar tum `node server.js` se server chala rahe ho, aur koi code change karte ho, to:

* Tumhe manually server **band kar ke dubara start** karna padta hai.

---

### ‚úÖ With `nodemon`:
* Tum just `npm run dev` likh do.
* Jab bhi tum server code (e.g., `server.js`, `routes`, `controllers`) mein kuch change karte ho...
* `nodemon` **automatically server ko restart** kar deta hai.

---

### üîß Example:
npm install nodemon --save-dev
```

Phir `package.json` mein:

```json
"scripts": {
  "dev": "nodemon server.js"
}
```

Ab jab bhi tum:

```bash
npm run dev
```

likhoge, `nodemon` tumhare liye server ko monitor karega.

---

### üí° Summary:

| Feature              | `node`      | `nodemon`        |
| -------------------- | ----------- | ---------------- |
| Auto-restart server? | ‚ùå No        | ‚úÖ Yes            |
| Best for development | ‚ùå Not ideal | ‚úÖ Perfect choice |

---

Agar tum development kar rahi ho (like EAD project), to `nodemon` use karna **bohot helpful** hota hai.

Bolo agar install krne mein help chahiye!
